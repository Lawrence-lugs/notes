Rings and Fields

Published

January 3, 2026

Abstract algebra is apparently the study of rings and fields plus groups…

Groups

Groups are sets with ONE operation satisfying closure (must stay within the group), associativity, identity and inverses.

Rings

Rings

Ring (German: Zahlring). A cycle of return. Sets that close back onto themselves on their operations.

Rings are a set with addition and multiplication. So like, a group + another operation.

It’s less restrictive than an algebra. A ring only has to be closed over itself. That is, a closed set on addition and multiplication. An algebra, on the other hand, always has to be defined over something*. It has to be a vector space \(A\) over \(K\) equipped with a multiplication that is compatible with the scalars.

Algebra cares about basis and dimension. Rings only care about “primeness” and factorization.

Why rings?

Commutative algebra is the study of rings. Rings were first delineated when the concept of ideals came about.

Rings have important common properties we take advantage of like unique factorization domains (a type of ring), and a bunch of other mouthful theorems like

  • Maximal Ideals are one-to-one with Fields (see later section) (\(M \text{ maximal} \iff R/M \text{ is a field}\))
  • \(R/\text{ker}(\phi)\cong \text{im}(\phi)\).
  • \(u \in R\) is a unit iff \(\text{Ideal}(u)=R\) (defines what a “unit” element is, like 1 or i. Probably useful for induction on weird sets too?)

Tl;DR, things close to rings (groups, commutative rings, rings with unique factorization, fields) are useful. Theorems that apply ONLY to rings themselves are a little abstract (heh, pun intended).

Ideals are subsets of rings (technically called a sublattice, probably because it’s a “discrete vector field” because it’s a linear combination of the ring elements)

Ideals can be factored, and can be prime.

What is an Ideal?

An ideal \(I\) is a subset of the ring that absorbs multiplication. If \(r \in R\) and \(x \in I\), then \(rx \in I\). Notation: \((a, b)\) is the ideal generated by \(a\) and \(b\). It implies the set of all linear combinations \(\{ra + sb \mid r, s \in R\}\).

Note

The Real ring \(\mathbb{R}\) has exactly two ideals:

  1. Zero ideal (0)
  2. Entire ring (1)

It is trivial. This is because when trying to define any other ideal, say (0.001), you accidentally become the (1) ideal because you always have a multiplative inverse.

Maximal vs Prime Ideals

Maximal Ideal: An ideal \(M\) is maximal if no other proper ideal \(J\) contains \(M\) (i.e., \(M\subset J\subset R\) is impossible). Prime Ideal: An ideal \(P\) is prime if for any \(a,b\) in the ring, if \(ab\in P\), then \(a\in P\) or \(b\in P\)

Vector spaces

Note

Vector spaces are

  • Abelian Groups - sets closed under one operation that is associative, commutative, has an identity and inverses.
  • Scalable by a Field - must be multipliable by some field that still keeps it closed.

Fields

A ring with division- that is, each element has a multiplicative inverse. \(ax=b\) is always solvable in a field. \(ax=b\) would be a diophantine equation if it’s on the ring \(\Bbb{Z}\), and can have no solutions (say, if \(a\) and \(b\) are coprime)

Difference between Ring, Field and Algebra.

A ring is a set with addition and multiplication closed over itself. It must also have additive inverses.

A field is a commutative ring (ring with commutative multiplication) with an inverse defined for every non-zero element. That is, it also has multiplicative inverses as opposed to just additive inverses.

All fields are rings.

An algebra can be defined over a field or a ring. You need:

  • To be “Over something” - Over a set with multiplication and addition (So at least a ring)
  • Scalars that when multiplied to the elements, stays within the set.
  • The scalars need not come from the “over” set. Why need the scalars? Scalars make it so that you can build all elements of the set from a few elements (a basis). This then gives access to the tools of linear algebra: determinants, traces, matrices, eigenstuff.

Nomenclature:

  • Algebras over Rings are special Modules (+free, meaning it has a basis)
  • Algebras over Fields are special Vector Spaces (+multiplication)

A few interesting things:

  • A ring can be an algebra over itself. So is a field an algebra over itself. These are trivial though.

Example: Finding all the Pythagorean Triples of Primitive Triples

\[ a^2 + b^2 = c^2\] \[ (a + ib)(a - ib) = c^2\]

We are now in the complex integer domain. Ring concept: \(\mathbb{Z[i]}\) (Gaussian integers- complex numbers of the form \(a + bi; a,b \in \mathbb{R}\)) is a unique factorization ring meaning there are prime elements and every non-prime element can be factorized into prime elements. This is the equivalent of the fundamental theorem of algebra for rings.

That is, they are either prime or composite. If the set of factors isn’t all prime yet, you can still decompose them.

This allows us to use factoring concepts for complex numbers.

Let \(x=(a+ib)\) and \(y=(a-ib)\)

Let’s assume that \(x\) and \(y\) are coprime. Then, they must necessarily also be squares.

Proof

… apparently this proof is more complex than I thought. It’s not a first principles proof…

If \(\gcd(a,b)=1\) and \(ab=n^2\), then \(a,b\) are also perfect squares requires some GCD properties

\(\gcd(a,b)^2 = \gcd(a^2,b^2,ab)\) requires Bezout’s Identity

Anyway with that we can do \[(a+ib)=(t+iv)^2\] then \[t^2-v^2+i2vt=a+ib\] then we equate real and imaginary (simple addition reduction) \[t^2-v^2=a\] \[b=2vt\] then for any integer \(v,t\), \(a\) and \(b\) will form a pythagorean triple.

Lastly, we assumed that \(x\) and \(y\) are coprime. Are they?

\(x\) and \(y\) are coprime

Assume that \(x\) and \(y\) are not coprime. Then with some \(\delta\) which is the common divisor of \(x\) and \(y\):

  • \(\delta\) divides \(x+y = (a+bi)+(a-bi) = 2a\)
  • \(\delta\) divides \(x-y = (a+bi)-(a-bi) = 2ib\)

\(c\) is always odd in a primitive triple which means \(c^2\) is odd and so \(x\) or \(y\) is odd. This means \(\delta\) cannot have a factor of 2.

So then \(\delta\) must be a common factor of \(a\) and \(b\). But it’s a primitive triple \(a^2+b^2=c^2\), so they can’t have a common factor.

This example didn’t actually require rings- just the unique factorization theorem from rings which applies to the Gaussian Integers. In this next one, we actually need rings.
